A fun and interactive card flipping memory game built to test and improve your concentration and memory skills. The objective of the game is to flip cards and find matching pairs by remembering their positions.

Players flip two cards at a time:

If the cards match, they remain face up.

If they don’t match, they flip back over after a short delay.

The game continues until all card pairs are successfully matched.

 Features:

Simple and intuitive gameplay

Smooth card flip animations

Randomized card positions on every game start

Tracks player progress (moves, time, or score — if implemented)

Responsive design for desktop and mobile devices

Technologies Used:

HTML

CSS

JavaScript

This project is perfect for beginners looking to practice DOM manipulation, event handling, and game logic, or for anyone who enjoys classic memory games.
